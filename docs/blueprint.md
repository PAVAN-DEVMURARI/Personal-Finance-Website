# **App Name**: FinPower

## Core Features:

- Dashboard Overview: Display total income, expenses, net balance, spending pie chart, and savings goals progress on a central dashboard.
- Expense Tracking: Enable users to add, categorize (Food, Travel, Rent, etc.), edit, and delete expenses, with weekly/monthly analytics.
- Income Tracking: Allow users to add income sources, track total income per month, and cumulative savings.
- Savings Goals Management: Let students set financial goals with target amounts, due dates, and manual savings updates to track progress.
- Financial Tips Feed: Rotate short, motivational financial tips to encourage better money habits.
- Gamified Engagement: Implement daily streaks for expense logging and badges for savings milestones to boost user engagement.
- Secure Authentication: Provide Google sign-in or email/password authentication to securely store user data. Includes a Reports page with a monthly summary of income vs expenses with options to export the data in formats like CSV or PDF.
- Investment Tracking: Track user investments across mutual funds, stocks, ETFs, crypto, etc. Display overall investment done in total in sections like monthly, yearly, etc.
- AI Financial Report and Feedback: Use AI that tracks all your investment and spending, provides a brief report at the end of the month, and gives feedback based on it.

## Style Guidelines:

- Primary color: Light teal (#74D9C5), inspired by the concept of financial growth and tranquility, promoting a sense of balance and well-being.
- Background color: Off-white (#F2F4F3), desaturated from the teal, providing a clean and calm backdrop that enhances readability.
- Accent color: Soft orange (#E8A36E), an analogous hue to teal that injects energy and optimism, perfect for CTAs and highlighting achievements.
- Headline font: 'Space Grotesk' sans-serif for headlines, and 'Inter' sans-serif for body text.
- Utilize simple, friendly, emoji-like icons for expense categories to make the interface intuitive and fun.
- Employ a sidebar navigation for key sections (Dashboard, Expenses, Goals) and cards for concise content display.
- Incorporate subtle animations for transactions and goal updates, providing delightful feedback to user interactions.
